# 🐜 Ant Cake Defense

> 플래시 시대의 고전 타워 디펜스를 모던 브라우저에서 재해석한 웹 게임.

좌측 상단 개미굴에서 기어나오는 개미들이 우측 하단의 케이크 8조각을 노립니다.
크레딧으로 타워를 건설·진화시켜 40 웨이브의 공세를 막아내세요.

**▶ [브라우저에서 바로 플레이](https://gogakaka.github.io/ant-cake-defense/)**

순수 HTML / CSS / JavaScript · 빌드 과정 없음 · 설치 불필요

---

## 🎯 게임 목표

- **개미는 승리 조건**: 케이크 8조각을 전부 개미굴로 가져가면 패배
- **플레이어는 생존**: 40 웨이브가 끝날 때까지 케이크 하나라도 지키면 승리
- 개미가 케이크를 들고 가는 **도중에 처치하면 케이크는 제자리로 복귀**

## 🎮 플레이 방법

1. **빈 칸 클릭** → 설치 확인 팝업이 뜨고, **"기본 타워 건설"** 버튼으로 설치 확정
2. **타워 클릭** → 진화/철거 메뉴가 열림
3. 크레딧은 적 처치와 웨이브 완료로 획득
4. 상단 HUD에서 **1x / 2x / 3x 배속**, **일시정지**, **다시 시작** 가능

### 단축키

| 키 | 동작 |
|---|---|
| `Space` | 일시정지 / 재개 |
| `F` | 배속 전환 (1x → 2x → 3x) |
| `R` | 게임 재시작 |
| `Esc` | 선택 해제 · 팝업 닫기 |

## 📈 웨이브 구성 (40 라운드)

- **1~10**: 튜토리얼 → 탱크 등장 → 첫 엘리트 · **웨이브 10에 여왕 1마리**
- **11~20**: 엘리트 본격화 · **웨이브 15 여왕 2** · **웨이브 20 여왕 3**
- **21~30**: 대규모 혼성 공세 · **웨이브 25 여왕 3** · **웨이브 30 대왕 1 + 여왕 1**
- **31~40**: 최종 폭주 · **웨이브 35 대왕 2** · **웨이브 40 대왕 5** (최종 보스 러시)

웨이브 사이에는 6초 준비 시간이 주어집니다.

## 🛠️ 로컬에서 실행

의존성 없이 **`index.html` 을 더블 클릭**하면 끝.

```bash
git clone https://github.com/gogakaka/ant-cake-defense.git
cd ant-cake-defense
# Windows
start index.html
# macOS
open index.html
# Linux
xdg-open index.html
```

원하면 로컬 서버로 띄워도 됩니다:

```bash
python -m http.server 8000
# → http://localhost:8000
```

## 🧱 파일 구조

```
ant-cake-defense/
├── index.html           # 캔버스 + HUD 오버레이
├── style.css            # 다크 톤 글래스 UI

│   └── game.js          # Game 클래스 + 메인 루프 + UI 연결
└── README.md
```

- **엔진 없음**: HTML5 Canvas 2D + vanilla JS (빌드 / 번들러 / 프레임워크 모두 미사용)
- **경로 탐색**: 40×20 그리드 위의 A* (바이너리 힙)
- **적 개체수**: 최대 300+ 동시 처리
