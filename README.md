# 🐜 Ant Cake Defense

> 플래시 시대의 고전 타워 디펜스를 모던 브라우저에서 재해석한 웹 게임.

좌측 상단 개미굴에서 기어나오는 개미들이 우측 하단의 케이크 8조각을 노립니다.
크레딧으로 타워를 건설·진화시켜 **70 웨이브**의 지수 급성장 공세를 막아내세요.

**▶ [브라우저에서 바로 플레이](https://gogakaka.github.io/ant-cake-defense/)**

순수 HTML / CSS / JavaScript · 빌드 과정 없음 · 설치 불필요

---

## 🎯 게임 목표

- **개미는 승리 조건**: 케이크 8조각을 전부 개미굴로 가져가면 패배
- **플레이어는 생존**: 70 웨이브가 끝날 때까지 케이크 하나라도 지키면 승리
- 개미가 케이크를 들고 가는 **도중에 처치하면 케이크는 제자리로 복귀**
- 타워는 경로를 **막지 않음** — 개미는 타워 위를 통과함. 배치는 순수 사거리 전략

## 🎮 플레이 방법

1. **빈 칸 클릭** → 설치 확인 팝업이 뜨고, **"기본 타워 건설"** 버튼으로 설치 확정 (50 크레딧)
2. **타워 클릭** → 진화(Lv.2 → Lv.5) / 철거(60% 환불) 메뉴
3. 크레딧은 적 처치 · 웨이브 완료 · **금광 타워**로 획득
4. 상단 HUD에서 **1x / 2x / 3x 배속**, **일시정지**, **다시 시작** 가능

### 단축키

| 키 | 동작 |
|---|---|
| `Space` | 일시정지 / 재개 |
| `F` | 배속 전환 (1x → 2x → 3x) |
| `R` | 게임 재시작 |
| `Esc` | 선택 해제 · 팝업 닫기 |

## 🏗️ 타워 진화 트리 (8경로 × 5단계)

```
               ┌─→ ⚡ 레이저     → 플라즈마    → 스톰        → 보이드
               │   빠른 연사 빔     2체인        3체인         5체인
               │
               ├─→ 🚀 미사일     → 핵탄두      → 시즈        → 오비탈
               │   소폭발          중폭발        대폭발        초대형 폭발
               │
               ├─→ ✦ 2중탄      → 트리플      → 쿼드        → 바라지
               │   2연발           3연발         4연발         5연발
               │
기본 타워 ────┼─→ ◎ 저격       → 대물저격    → 레일건      → 광자포
 50 크레딧     │   초장거리 단발   더 먼 거리    전자기 포     맵 전역 사정거리
               │
               ├─→ ❄ 얼음       → 서리        → 빙결        → 절대영도
               │   30% 둔화 오라   45% 둔화      60% 둔화      72% 둔화
               │
               ├─→ 🔥 화염       → 불꽃        → 용광로      → 지옥불
               │   근거리 강지딜   더 뜨거움     광역 섬멸     모두 태움
               │
               ├─→ ✺ 오라       → 지휘본부    → 전략기지    → 최고사령부
               │   주변 +12%      +22%          +32%          +48%
               │
               └─→ 💰 금광       → 금고        → 조폐국      → 왕립조폐국
                   +1/s 크레딧    +2.5/s        +5/s          +9/s
```

### 경로별 성격

| 경로 | 특성 | 역할 |
|---|---|---|
| ⚡ **레이저** | 빠른 연사 + 연쇄 번개(2→5마리) | 엘리트/스웜 혼합 대응 |
| 🚀 **미사일** | 느린 폭발탄 + 광역 피해 | 탱크/엘리트 밀집 제압 |
| ✦ **2중탄** | 다연발 (2→5발) · 안정적 단일 DPS | 보스전 등 지속 딜 |
| ◎ **저격** | 초장거리 (최대 9셀) · 최고 단발 피해 | 맵 커버형 포지션 |
| ❄ **얼음** | 수동 오라 · 둔화 + 약한 지속 피해 | 크라우드 컨트롤 |
| 🔥 **화염** | 짧은 근거리 · 초당 최대 120 지속 딜 | 병목 근접 섬멸 |
| ✺ **오라** | 공격 X · 주변 타워 DMG·연사 +48%까지 | 지원형 배치 퍼즐 |
| 💰 **금광** | 공격 X · 초당 최대 9 크레딧 자동 | 장기 경제 투자 |

## 🐞 적 종류

| 타입 | 1웨이브 체력 | 속도 | 보상 | 특징 |
|---|---:|---:|---:|---|
| 기본 개미 | 14 | 55 | 3 | 웨이브마다 체력 추가 +4% 복리 |
| 빠른 개미 | 9 | 92 | 3 | 약하지만 스피디 |
| 탱크 개미 | 46 | 36 | 9 | 느리고 질김 |
| 엘리트 개미 | 90 | 48 | 18 | 후반 단골 |
| 👑 여왕 (Boss Mini) | 320 | 30 | 60 | 보라 오라 + 3스파이크 왕관 |
| 💀 대왕 (Boss Big) | 780 | 22 | 180 | 붉은 오라 + 5스파이크 황금 왕관 |

> 전 개체의 이동 속도·HP는 **웨이브마다 지수 성장** (공비 HP 1.07, 속도 1.018)

| 웨이브 | HP 배율 | 속도 배율 |
|---:|---:|---:|
| 10 | 1.84× | 1.18× |
| 20 | 3.62× | 1.41× |
| 30 | 7.61× | 1.68× |
| 40 | 14.97× | 2.01× |
| 50 | 29.46× | 2.41× |
| 60 | 57.95× | 2.87× |
| **70** | **113.99×** | **3.43×** |

또 개체마다 속도 ±15% 편차와 경로 셀 내부 랜덤 목표점이 적용돼, 개미 떼가 **마칭이 아닌 유기적 흐름**으로 움직입니다.

## 📈 웨이브 구성 (70 라운드)

- **1~10**: 튜토리얼 · 탱크 등장 · 첫 엘리트 (10웨이브 여왕 1)
- **11~20**: 엘리트 본격화 · 여왕 2~3마리
- **21~30**: 혼성 공세 확대 · 30웨이브 **대왕 1 + 여왕 2** 등장
- **31~40**: 최종 보스 러시 대비 · 40웨이브 **대왕 8** 러시
- **41~50**: 물량·체력 한계치 돌파 · 대왕 10마리 투입
- **51~60**: 극한 압박 · 보스 풀업 타워 없이는 버티기 어려움
- **61~70**: **종말 구간** · 70웨이브엔 **대왕 50 + 여왕 20** + 일반 개미 1,000+

웨이브 사이 준비 시간은 **4초**. 처치·격퇴 보너스(20 + 웨이브×5)로 점진적 크레딧 제공.

## 🛠️ 로컬에서 실행

의존성 없이 **`index.html` 을 더블 클릭**하면 끝.

```bash
git clone https://github.com/gogakaka/ant-cake-defense.git
cd ant-cake-defense
# Windows
start index.html
# macOS
open index.html
# Linux
xdg-open index.html
```

원하면 로컬 서버로 띄워도 됩니다:

```bash
python -m http.server 8000
# → http://localhost:8000
```

## 🧱 파일 구조

```
ant-cake-defense/
├── index.html           # 캔버스 + HUD 오버레이
├── style.css            # 라이트 글래스 UI
├── src/
│   ├── constants.js     # 그리드, 팔레트, 32개 타워 정의, 70 웨이브 데이터
│   ├── utils.js         # 수학/좌표 변환 헬퍼
│   ├── pathfinding.js   # A* + 바이너리 힙 (타워 비차단)
│   ├── entities.js      # Ant / Projectile / Particle / Tower / Cake
│   └── game.js          # Game 클래스 + 메인 루프 + HUD/팝업
└── README.md
```

- **엔진 없음**: HTML5 Canvas 2D + vanilla JS · 빌드/번들러/프레임워크 모두 미사용
- **해상도**: 1600×880 (40×20 그리드 · 셀당 40px · HUD 80px)
- **반응형**: 뷰포트에 맞춰 비율 유지하며 스케일
- **경로 탐색**: 40×20 그리드 위 A* — 타워는 개미를 **막지 않음**
- **성능**: 동시 개미 1,000+ · 파티클/프로젝타일 실시간 처리
